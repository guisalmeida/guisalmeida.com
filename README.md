## Start
```
git clone git@github.com:GuiSAlmeida/guisalmeida.com.git
cd guisalmeida.com
yarn install # or npm install

#install gatsby cli
yarn global add gatsby-cli # or npm install -g gatsby-cli

# up server at http://localhost:8000/
yarn start # or npm run start
```
