## Start
```bash
    git clone git@github.com:GuiSAlmeida/guisalmeida.com.git
    cd guisalmeida.com
    yarn install # or npm install

    #install gatsby cli
    npm install -g gatsby-cli

    # up server at http://localhost:8000/
    gatsby develop 
```
